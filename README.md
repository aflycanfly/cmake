# Linux下开发C/C++

> 此文档是在Linuxx下C/C++开发的工具文档

在文档开始之前，我们先安装编译工具与调试工具

```shell
sudo apt install -y build-essential gcc g++ gdb cmake
```

安装验证，分别执行以下每一行命令，如果成功显示版本号，则代表安装成功

```shell
gcc --version
g++ --version
gdb --version
cmake --version
```
